
Курсовой проект по дисциплине «Базы данных».

## Описание

Приложение предназначено для учета поставщиков, деталей, цен, поставок и поставок_деталей. Реализована клиент-серверная архитектура с использованием Java, MySQL и шаблона проектирования MVC.

## Используемые технологии

- Java
- Swing
- MySQL
- JDBC
- Git
- MVC

## Возможности

- регистрация пользователей;
- авторизация с хешированием паролей (SHA-256);
- проверка сложности пароля;
- ролевая модель доступа (Admin, Manager, User);
- CRUD-операции для основных сущностей;
- работа с базой данных через JDBC.

## Структура проекта

```
src/
├── controller/
├── database/
├── model/
├── repository/
├── view/
└── Main.java
```

## Запуск

1. Создать базу данных MySQL.
2. Выполнить SQL-скрипт создания таблиц.
3. Настроить параметры подключения в `DBConnection.java`.
4. Запустить `Main.java` или использовать собранный JAR-файл.
